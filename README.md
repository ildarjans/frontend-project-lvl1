### About
Package has five games:  
- [**brain-even**](#even) 
- [**brain-calc**](#calc)  
- [**brain-gcd**](#gcd)
- [**brain-prime**](#prime)
- [**brain-progression**](#progression)

### Installation   

```
make install
```

```
make brain-games
```
