### About
Package has five games:  
- [**brain-even**](#even) 
- [**brain-calc**](#calc)  
- [**brain-gcd**](#gcd)
- [**brain-prime**](#prime)
- [**brain-progression**](#progression)

### Installation   

```
make install
```    

### Get started

```
$ make brain-games
                                                        
> Welcome to the Brain Games!                                                     
> May I have your name? Frodo                                                     
> Hello, Frodo! 
```
